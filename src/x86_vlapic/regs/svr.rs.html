55/// Spurious-Interrupt Vector Register using MMIO.
56/// - Address: FEE0 00F0H
57/// - Value after reset: 0000 00FFH
58///
59/// A special situation may occur when a processor raises its task priority to be greater than or equal to the level of the interrupt for which the processor INTR signal is currently being asserted.
60/// If at the time the INTA cycle is issued, the interrupt that was to be dispensed has become masked (programmed by software), the local APIC will deliver a spurious-interrupt vector.
61/// Dispensing the spurious-interrupt vector does not affect the ISR, so the handler for this vector should return without an EOI.
62pub type SpuriousInterruptVectorRegisterMmio = ReadWrite<u32, SPURIOUS_INTERRUPT_VECTOR::Register>;
63
64/// A read-write copy of Spurious-Interrupt Vector Register (FEE0 00F0H).
65///
66/// This behaves very similarly to a MMIO read-write register, but instead of doing a
67/// volatile read to MMIO to get the value for each function call, a copy of the
68/// register contents are stored locally in memory.
69pub type SpuriousInterruptVectorRegisterLocal =
70    LocalRegisterCopy<u32, SPURIOUS_INTERRUPT_VECTOR::Register>;
